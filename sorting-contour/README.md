# Sorting Contours

## our goal:
- sorting contour with 4 method
    - left-to-right
    - top-to-buttom
    - right-to-top
    - buttom-to-top

## Run the code
```
$ python sorting-contour.py --image image_01.png --method top-to-buttom
```
